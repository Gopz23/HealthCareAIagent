
# Healthcare Multi-Agent AI System

## Overview
AI-powered healthcare assistant using multi-agent architecture and RAG.

## Features
- Multi-agent design (Risk, Recommendation, Safety)
- Retrieval-Augmented Generation (FAISS)
- Conversational UI (Gradio)
- Safety guardrails

## Architecture
(User → Agents → RAG → LLM → Response)

## Tech Stack
- Transformers (FLAN-T5)
- Sentence Transformers
- FAISS
- Gradio
